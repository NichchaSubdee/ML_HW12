Assignment 12

Topic: Propensity Score Pipeline

This assignment applies elastic net as an additional method for estimating propensity scores using the same dataset from class (nmes_data). After propensity scores are estimated, individuals are matched and the association between smoking and respiratory outcomes is evaluated. The elastic net results are compared with logistic regression and random forest in terms of the number of individuals retained, covariate balance, and the estimated effect. The final model is then selected based on overall performance.

Dataset

The data come from a subset of the 1987 U.S. National Medical Expenditures Survey. The exposure is eversmk, which indicates whether a person has ever smoked, and the outcome is lc5, which indicates lung cancer, laryngeal cancer, or COPD.

The covariates used to estimate the propensity score are age, sex, race, education, seatbelt use, marital status, census region, and poverty status.

Random forest propensity score

Next, we estimated the propensity score using random forest.

set.seed(123)

feat.count <- c((ncol(nmesdata) - 4), (ncol(nmesdata) - 4) / 2, sqrt(ncol(nmesdata) - 4))
grid.rf <- expand.grid(mtry = feat.count)
tree.num <- seq(100, 500, by = 200)

results.trees <- list()

for (ntree in tree.num) {
  set.seed(123)
  rf.train <- train(
    eversmk ~ LASTAGE + MALE + educate + beltuse + POVSTALB + marital + RACE3 + SREGION,
    data = nmesdata,
    method = "rf",
    metric = "Accuracy",
    tuneGrid = grid.rf,
    importance = TRUE,
    ntree = ntree
  )
  index <- toString(ntree)
  results.trees[[index]] <- rf.train$results
}

output.trees <- bind_rows(results.trees, .id = "ntrees")
best.tune <- output.trees[which.max(output.trees[, "Accuracy"]), ]
